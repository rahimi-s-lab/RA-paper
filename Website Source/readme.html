Instructions on how to use the web application

A user should input a text file (.txt) into the webserver. In the text file, every single line should include just 9 numbers, which are separated by commas. The numbers, also, should be in the range of [0, 1]. The table below describes the meaning of these 9 numbers.

Number

Name

Explanation

1

Rest pain

Pain is one the most common symptoms in patients with RA.

2

Morning stiffness

This symptom occurs commonly among patients with RA and leads their life quality to be reduced.

3

Symmetry of joint infection

Joint involvement, which occurs symmetrically in the patients with RA, is one of the hallmarks of the RA.

4

Redness

It is possible that in patients with RA, due to inflammation, joints become red and even warmer.

5

Body pain

Patients with RA usually experience moderate as well as persisting pain in their body.

6

Swelling

One of the symptoms of RA is synovitis, which can breed swelling in the joints.

7

Rheumatoid factor (RF)

This test determines the amount of RF in one’s blood.

8

Erythrocyte sedimentation rate (ESR)

It is a test which is able to determine the severity of inflammation inside a body.

9

Anti-cyclic citrullinated peptide antibody test (Anti-CCP)

It is a lab test which is positive in 57% to 66% of patients suffering from RA.

 

As an example, a line of the text file can be like this:

0.5, 0.5, 0.5, 0.3, 0.3, 0.3, 0.3, 0.5, 0.5

The text file has no laminations regarding the number of lines. You can find a valid text file here. After submitting your file, the webserver will automatically do calculations based on our proposed algorithm and will export a text file, in which you can find every line’s associated severity level.

Disclaimer: This tool is developed based on the limited number data, so it should not be used as a replacement for your physician. Final decision will be made by physicians.
